/****** Translate Class ****** * This class allows you to translate between multiple languages. * ****** Example of using this class ****** First, this checks for the plugin of your desired language. Translate::whatLanguage("French"); Searches for the plugin "LanguageFrench" If it exists, it runs everything through there. Use of the plugin is like this: $lang = new Translate(); $lang->Welcome // Prints "Welcome" if there is no substitude in the language provided $lang->Yes // Prints "Si" (spanish equivalent) if the substitude is provided by the language You can choose to disable the lookups when you launch Translate. Such as $lang->noTranslate(); ****** Methods Available ****** * $object = Translate::setLanguage(); */
