Search::sql(); 2. Populate the search_primary table with Search::setKeyword(....); 3. Add Search::searchbar() 4. Add /ajax/search-bar.php 5. Scripts::defaultHashtagSearch() ****** Example of using the class ****** ****** Methods Available ****** * $results = Search::run($query, [$table]); * $results = Search::runUsers($query, $baseURL); * * Search::recordSearch($keyword); * Search::setKeyword($keyword, $url, $minChars = 1, $sortOrder = 1); * Search::deleteKeyword($keyword); * * Search::searchBar($name, $scriptName, $placeholder); * Search::searchBarUserHandle(); */
